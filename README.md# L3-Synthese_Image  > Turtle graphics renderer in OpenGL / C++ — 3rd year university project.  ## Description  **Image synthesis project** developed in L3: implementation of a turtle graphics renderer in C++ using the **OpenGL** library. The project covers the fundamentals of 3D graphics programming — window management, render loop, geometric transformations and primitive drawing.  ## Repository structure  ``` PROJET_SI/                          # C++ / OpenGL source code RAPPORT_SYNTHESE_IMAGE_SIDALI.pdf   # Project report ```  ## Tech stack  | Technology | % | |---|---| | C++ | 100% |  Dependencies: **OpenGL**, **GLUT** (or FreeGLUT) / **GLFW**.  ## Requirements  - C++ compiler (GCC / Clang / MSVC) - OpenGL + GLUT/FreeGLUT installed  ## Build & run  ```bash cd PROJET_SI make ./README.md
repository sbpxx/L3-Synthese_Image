# L3-Synthese_Image

> Turtle graphics renderer in OpenGL / C++ — 3rd year university project.

## Description

**Image synthesis project** developed in L3: implementation of a turtle graphics renderer in C++ using the **OpenGL** library. The project covers the fundamentals of 3D graphics programming — window management, render loop, geometric transformations and primitive drawing.

## Repository structure

```
PROJET_SI/                          # C++ / OpenGL source code
RAPPORT_SYNTHESE_IMAGE_SIDALI.pdf   # Project report
```

## Tech stack

| Technology | % |
|---|---|
| C++ | 100% |

Dependencies: **OpenGL**, **GLUT** (or FreeGLUT) / **GLFW**.

## Requirements

- C++ compiler (GCC / Clang / MSVC)
- OpenGL + GLUT/FreeGLUT installed

## Build & run

```bash
cd PROJET_SI
make
./synthese
```
